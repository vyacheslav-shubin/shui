## О документе

**2024.06.08** — Данный документ является логом разработки. Некоторые идеи, описанные в его начале, переосмыслены. Многие блоки Marlin заменены на собственные (в т.ч. маппинг портов). Ядро GRBL оставлено практически без изменений.

**С 2021.04.09** файлы `*_config.gcode` заменены на `@*.gcode` — файлы, начинающиеся с `@`, рассматриваются прошивкой как конфигурационные.

## Состав прошивки

| Файл | Назначение |
|------|-------------|
| `robin_nano35.bin` | Код прошивки |
| `RESDUMP.BIN` | Картинки, шрифты, прочие ресурсы |
| `cfg/@printer.gcode` | Конфигурация |
| `cfg/@pid.gcode` | Демо настройки PID (виджет с динамикой температуры) |
| `preview-test` | Тестовые файлы для проверки превью |
| `SHUIWIFI.BIN` | Прошивка модуля WiFi |

Файлы для прошивки должны находиться **в корне флешки**.

### Только для MKS ROBIN 1.1 с модулем WiFi

- **ROBIN11.BIN** — маркер типа платы при прошивке WiFi. Для 1.1 во время прошивки файл должен быть в корне флешки.
- MKS ROBIN 1.1 и 1.2 используют разные порты для перевода WiFi в режим прошивки.

**Важно:** `ROBIN11.BIN`, `SHUIWIFI.BIN`, `RESDUMP.BIN` не переименовывать в нижний регистр (Marlin 2 не поддерживает удаление/переименование длинных имён; к `robin_nano35.bin` это не относится).

### Дополнительно

- **cura.zip** — плагин для CURA и сценарий создания превью в G-code.
- **ShuiPreview.py** — скрипт для CURA (превью SHUI и MKS, размеры 100×100 и 50×50). Устанавливается в `<Cura configuration>/scripts` (Help → Cura configuration). Рекомендуется удалить плагин MKS, чтобы не путать «Save to File» и «Save as TFT».

**Плагин «Send to SHUI WIFI»:** в адресной строке — IP принтера, в имени файла — не более 8 символов формата 8.3 (латиница, цифры, часть знаков). Нарушение формата может привести к ошибке загрузки.

## EEPROM и откат прошивки

Использование памяти EEPROM отличается от версии MKS. При обратной прошивке нужна полная инициализация:

- `/mks_pics`
- `/mks_fonts`
- `robin_nano35.bin`
- `robin_nano_config.txt`

---

## Особенности

- Концевики: **X-min, Y-max, Z-min**
- Настройки выполняются через G-code. После прошивки конфиг запускать вручную. Подтверждение: **settings stored** в статусной строке.
- Специфичные команды прошивки: **`M117 SHUI …`**

## Краткая сводка по версиям (лог разработки)

Ниже — сжатая хронология изменений. Полные описания команд и опций см. в соответствующих разделах и в `gcode_info.txt`.

### 2021-02 — 2021-04

- Печать, виджеты файлов, конфигурация концевиков, размера стола, пресеты температур.
- Виджеты: перемещение, печать, парковка, управление питанием, уровень стола, консоль G-code, превью.
- Поддержка сниппетов, LINEAR ADVANCE, BlTouch (Z-MIN, затем Z_MAX), бакап, M2000+.

### 2021-05 — 2021-07

- Удалён M117 SHUI, введены M2000+.
- BlTouch, автоуровень (M2029), бакап MBL, G10/G11, Fat time.

### 2021-08 — 2021-10

- Sapphire Plus, два мотора Z, Telegram-бот, маппинг портов, CoreXY/CoreYX, BfTouch.
- Профили конфигурации, M2017 (UI), настраиваемые кнопки, QOI, виджет Cooking (йогурт-таймер).

### 2021-11 — 2022-01

- TMC2209 (UART), Pt1000, сенсоры, удаление файлов с длинными именами, новый FAT-драйвер.

### 2022-02 — 2022-06

- Переключаемая кинематика, RN13 WiFi, вентиляторы (SoftPWM, медвежий режим), M73, смена слоя.

### 2022-07 — 2022-12

- Input shaping, полигональная интерполяция сетки стола, G29 по области модели, телеметрия, TG-диск, ссылки на хранилища.

### 2023 — 2024

- Trinamic (UART, детектор), маппинг портов, лазер/гравер, коррекция ромба, M2036 (триггеры температуры), виджет Cooking.

### 2024-07 — 2025

- Телеметрия (сервер/напрямую в TG), отправка файлов в канал с подписью print/save/link, M0 с паузой (P/S/M), триггеры и действия по температуре, виджет Cooking (M2006 N33).

### 2025-01 — 2025-07

- MicroProbe, дублирование моторов X/Y, «толчок» при выгрузке, RN6, лазер на PA8, пресеты скорости (F1–F4) для лазера/гравера, запрет старта УП без парковки.

## Известные проблемы

- **Robin Nano 1.1** — не прошивает плату WiFi: откатиться до версии сентября, прошить WiFi, затем вернуть текущую версию.
- После загрузки файла определённого размера по сети возможно сообщение «Ошибка протокола обмена» (в обновлениях исправлялось).
- Кнопка выключения питания не работает — задать сниппет:  
  `M2005 S16|M81C` или `M2005 S16|M140S0|M104S0|M81C`

---

## Важные замечания по обновлениям

- Перед установкой тестовых/критичных обновлений **обязательно делать backup**.
- Если в примечании указано обновление **RESDUMP.BIN** или **SHUIWIFI.BIN** — их нужно обновить.
- **ESP32:** с некоторого момента старые прошивки WiFi могут быть несовместимы; при указании в примечании — обновление модуля обязательно. DEV-версия модуля имела ограничение по времени использования (30/90 дней).
- Файл **RESET.BIN** в корне флешки при включении сбрасывает настройки до умолчаний.
